# QTTC 2025 锦标赛赛程网页

## 项目简介

这是一个展示"2025年'觚棻'杯暨丘班乒乓球锦标赛"(QTTC 2025)完整赛程的网页应用。网页以响应式设计呈现小组赛和淘汰赛的对战信息、比分详情以及选手晋级路径。

## 功能特点

### 核心功能
- **完整赛程展示**：包括6个小组（A-F组）的小组赛和全部淘汰赛阶段
- **数据导入功能**：支持从JSON文件导入比赛数据，覆盖现有数据
- **响应式设计**：适配桌面端和移动端设备，手机端建议横屏以获得最佳体验
- **交互式界面**：点击比赛卡片可查看详细的比分信息
- **晋级路径可视化**：展示选手从小组赛到决赛的完整晋级路线

### 数据显示
- 小组赛采用卡片式展示，按A-F组标签页切换
- 淘汰赛按阶段（八分之一决赛、四分之一决赛、半决赛、决赛/铜牌赛）分页显示
- 实时计算并显示小组排名和选手晋级状态
- 显示种子选手信息（1-8号种子）
- 奖牌榜展示冠、亚、季军及相应种子位

### 数据详情
- 比赛时长、暂停和医疗暂停等事件记录
- 每场比赛的详细比分（总比分和各局小分）
- 弃赛和轮空比赛的特别处理
- 总得分统计

## 使用方法

### 直接使用
1. 直接打开`index.html`文件即可使用
2. 首次打开会显示预设的比赛数据

### 数据导入
1. 点击左侧边栏的"从JSON文件导入"按钮
2. 选择符合格式要求的JSON文件
3. 确认覆盖现有数据
4. 页面将自动刷新显示新数据

### 查看比赛详情
点击任意比赛卡片（轮空比赛除外），弹出模态框显示详细比分和比赛信息。

### 移动端使用建议
- 竖屏时会有横屏提示，建议横屏以获得更好的视觉效果
- 在窄屏设备上，标签页和比赛卡片会自动适配

## 数据结构说明

### 比赛数据格式
网页使用两个主要数据结构：

```javascript
// 1. 比赛数据
matchData = {
  "groupStage": [ // 小组赛数组
    {
      "stage": "A组", // 比赛阶段
      "matchNum": 1, // 场次编号
      "players": "选手A vs 选手B", // 选手对阵
      "score": "3-2", // 总比分
      "winner": "选手A", // 胜者
      "status": "completed", // 状态：completed, default, bye
      "details": {
        "sets": ["11-9", "9-11", "11-6", "8-11", "11-7"], // 各局比分
        "duration": 38, // 比赛时长（分钟）
        "timeout": [true, false], // 暂停情况（选手A, 选手B）
        "medical": [false, false], // 医疗暂停情况
        "notes": "比赛描述"
      }
    }
    // ... 更多比赛
  ],
  "knockoutStage": [ // 淘汰赛数组
    // 结构与小组赛类似
  ]
};

// 2. 选手数据
playerData = {
  "选手姓名": {
    "seed": 1, // 种子位（1-8或null）
    "group": "A组" // 所属小组
  }
  // ... 更多选手
};
```

### JSON导入文件格式
导入的JSON文件应包含完整的数据结构：
```json
{
  "matchData": {...},
  "playerData": {...}
}
```

## 自定义和扩展

### 修改样式
- 所有样式内嵌在HTML文件的`<style>`标签中
- 使用CSS变量和响应式设计，便于调整配色和布局
- 主要颜色方案：
  - 主色调：深蓝色 (#1a237e)
  - 强调色：橙色 (#ff5722)
  - 状态色：绿色（已完成）、橙色（未开始）、红色（弃赛）

### 添加功能
- 本地存储：数据会自动保存到浏览器的localStorage
- 模态框：比赛详情弹出窗口为只读展示
- 响应式断点：768px（平板）和420px（手机）

### 浏览器兼容性
- 支持现代浏览器（Chrome, Firefox, Safari, Edge）
- 使用ES6 JavaScript特性
- 包含移动端触摸滚动优化

## 注意事项

1. **数据持久性**：数据存储在浏览器本地，清除浏览器数据会丢失导入内容
2. **文件导入**：仅支持JSON格式文件，导入前会验证数据结构
3. **移动端体验**：在窄屏设备上，部分表格和标签页区域支持横向滚动
4. **种子选手**：种子位1-8号有特殊样式标记，非种子选手无标记
5. **特殊比赛状态**：
   - 弃赛：显示为红色边框，比分为3-0 (11-0, 11-0, 11-0)
   - 轮空：显示为灰色背景，无详细比分
   - 已完成：绿色边框，显示完整比分
